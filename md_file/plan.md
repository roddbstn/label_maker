# 라벨 메이커 (Label Maker) 기획서

## 📌 한 줄 요약
> "일일이 입력할 필요 없는 정부문서화일 라벨 텍스트 생성기"

---

## 🎯 해결하는 문제

### 현재 상황
- **사용 라벨:** 폼텍 라벨 3629
- **사용 빈도:** 주 1~2회 라벨지 출력
- **현재 방식:** 한글(.hwp) 파일 양식에 매번 텍스트를 직접 입력하여 출력

### 문제점

| 문제 | 상세 내용 |
|------|----------|
| 가독성 혼란 | 한 페이지에 두 쌍의 라벨이 있어 구분이 어려움 |
| 3가지 라벨 혼재 | ① 표지용(제목/연도/기관명) ② 측면용(길이가 긴 라벨) ③ 파일 내부 우상단용 |
| 더블체크 피로 | 출력 전 내용 확인 + 크기 적합성 확인 필요 |
| 휴먼에러 | 확인해도 실수 발생 → 재수정 → 재출력의 반복 |

### 핵심 페인포인트
1. **'노가다'의 온상:** 글자가 넘치면 일일이 장평/자간을 수동 조절해야 함
2. **데이터 연동 불가:** 엑셀 명단을 100개 만들려면 100번 복붙해야 함

---

## 💡 솔루션 개요

### 핵심 기능
**'자동 레이아웃 및 텍스트 피팅'** - 텍스트 입력 시 각 3개 영역별 크기에 맞게 자동 조절

### 작동 방식
```
입력 → 처리 → 출력 (→ 필요시: 수정 → 재처리)
```

### 입력 필드
| 구분 | 필드명 | 필수 여부 |
|------|--------|----------|
| 1 | 라벨지 제목 | ✅ 필수 |
| 2 | 생산연도 | ✅ 필수 |
| 3 | 부서명 | 선택 |
| 4 | 분류번호 | 선택 |
| 5 | 보존기간 | 선택 |
| 6 | 관리번호 | 선택 |

### 최종 산출물
- **수정/더블체크 불필요한 출력 준비된 라벨지 PDF 파일**

## 📊 시장성 분석

### 예상 타겟 (우선순위 순)
1. 공공기관 및 관공서 ⭐
2. 사회복지기관 (요양원, 아동센터, 아동보호전문기관, 치료센터, 쉼터 등)
3. 교육 기관 (학원, 학교, 대학)
4. 법무/세무/회계 법인
5. 건설/건축 설계 사무소
6. 일반 기업 지원 부서 (인사, 경영지원, 총무, 재무)
7. 금융권 (은행, 보험사, 증권사 지점)
8. 방송국

### 잠재적 사용자 규모
**약 65만 ~ 80만 명**
> 대한민국에서 "정부문서화일(황대일 파일) 규격"을 사용하여 주기적으로 라벨을 뽑아야 하는 실무자 총합

### 경쟁 환경
- **경쟁 서비스:** 없음
- **현재 대안:** 인터넷에 돌아다니는 '3629 한글 양식.hwp' 파일 다운로드 후 수동 편집

### 차별화 포인트

| 기능 | 설명 |
|------|------|
| 🌐 **No Install** | 인터넷만 되면 바로 접속 (관공서/기업 보안 환경 최적화) |
| 🎯 **Smart Fitting** | "대전광역시아동보호전문기관" 같은 긴 부서명도 자동 피팅 ⚡ **킬러 기능** |
| 📊 **Excel to Print** | 엑셀 업로드 → 대량 라벨 1초 만에 생성 |
