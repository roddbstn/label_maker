# 인스타그램 마케팅 콘텐츠 기획안 (Label Maker)

라벨 메이커의 핵심 가치인 **'업무 효율성'**과 **'시각적 만족감'**을 강조하여 사회복지사, 공무원, 사무직 종사자들을 타겟팅하는 마케팅 전략입니다.

---

## 1. 릴스(Reels) 시나리오: "1분 요약 업무 꿀팁"

### [시나리오 A: 비교형] "한글 작업 vs 라벨 메이커"
- **영상 구성 (15초)**:
    - **0~3초**: 한글(HWP)에서 글자 크기 조절하며 씨름하는 답답한 모습 (흑백 처리, 스트레스 이모지)
    - **3~5초**: "아직도 수동으로 맞추세요?" 자막과 함께 화면 전환
    - **5~12초**: 라벨 메이커 웹사이트에 접속해 제목 입력 → 자동으로 폰트 크기 줄어드는 '스마트 피팅' 시연 (컬러, 경쾌한 BGM)
    - **12~15초**: 'PDF 저장' 클릭 후 깔끔한 결과물 노출 + "프로필 링크 클릭!" 자막
- **BGM 추천**: 템포가 빠르고 변화가 명확한 비트

### [시나리오 B: 만족감형] "Oddly Satisfying Label Making"
- **영상 구성 (10초)**:
    - 아주 긴 제목(예: '2024년도 상반기 찾아가는 복지상담소 운영 관련 서류철')을 복사해서 붙여넣기 함.
    - 입력하자마자 칸에 맞춰 글자가 '스르륵' 줄어드는 장면을 클로즈업.
    - 마지막에 프린터에서 출력물이 깔끔하게 나오는 실물 촬영 컷.
- **포인트**: 자막 없이 시각적인 변화에만 집중하여 몰입감 유도.

## 2. 카드뉴스(Carousel) 기획: "실무자만 아는 라벨 작업의 고충"

| 슬라이드 | 이미지 내용 | 핵심 메시지 |
|:---:|:---|:---|
| **#1** | 파일철 라벨링 하는 지친 직장인 일러스트 | "라벨 하나 만들려고 폰트 1pt씩 줄여본 사람 손! 👋" |
| **#2** | 한글 양식의 복잡한 표와 칸 | "장평, 자간 조절... 우리가 이런 거 하려고 취업했나요? 😭" |
| **#3** | 라벨 메이커 UI 스크린샷 | "이제 텍스트만 넣으세요. 나머지는 저희가 합니다." |
| **#4** | 스마트 피팅 기능 강조 (Before/After) | "제목이 길어도 걱정 NO! 자동으로 최적의 크기를 찾습니다." |
| **#5** | PDF 저장 및 폼텍 3629 완벽 호환 | "폼텍 3629 규격 그대로, 클릭 한 번에 PDF 생성까지." |

## 3. 추천 캠페인 전략

### 핵심 해시태그
```text
#라벨메이커 #직장인꿀팁 #사회복지사 #공무원업무 #행정직 #업무효율 #서류정리 #파일링 #황대일파일 #정부문서화일 #사무용품 #라벨링
```

### 소통 전략
- **댓글 반응**: "한글 작업하다 빡쳐서 들어왔습니다" 같은 댓글에 "이제 해방되세요! 🎉" 와 같이 공감형 답글 작성.
- **스토리 활용**: "오늘 하루 라벨 작업 몇 개 하셨나요?" 설문 조사를 통해 니즈 파악.
