## 3. 대체재 (서비스 없이 해결하는 방법)

### 방법 1: 기존 HWP 파일 재활용
> "작년에 쓰던 파일 복사해서 연도만 바꿔요"

- **현실:** 대부분의 실무자가 사용하는 방법
- **한계:** 새 분류/새 부서/긴 제목 입력 시 장평 조절 노가다

### 방법 2: 신입/공익/실습생에게 시키기
> "이런 건 신입한테 시키면 돼"

- **현실:** 조직 내 업무 분배로 해결
- **한계:** 인력 부재 시 본인이 해야 함, 인수인계 문제

### 방법 3: 손글씨로 작성
> "급하면 볼펜으로 그냥 써요"

- **현실:** 소량/급한 경우 사용
- **한계:** 미관상 문제, 공식 문서에 부적합

### 방법 4: 안 하고 버팀
> "라벨 없어도 파일 찾는 데 문제 없어요"

- **현실:** 내부 문서는 라벨 없이 관리하기도 함
- **한계:** 감사 대비 시 지적 가능, 외부 제출 문서는 불가

## 5. 블루오션 기회

### 경쟁사들이 놓치고 있는 영역

| 기회 영역 | 설명 | 중요도 |
|----------|------|--------|
| **🎯 정부문서화일 3629 전용 웹 서비스** | 폼텍은 범용, 라벨스페이스는 아이라벨 중심. 3629 전용 웹 도구 없음 | ⭐⭐⭐⭐⭐ |
| **🔤 자동 텍스트 피팅** | "대전광역시아동보호전문기관"을 넣어도 알아서 맞춰주는 기능 없음 | ⭐⭐⭐⭐⭐ |
| **📊 엑셀 → 대량 PDF 원클릭** | 폼텍은 되지만 설치 필요. 웹에서 바로 되는 서비스 없음 | ⭐⭐⭐⭐ |
| **📱 모바일 대응** | 현장에서 스마트폰으로 라벨 제작 → 공유 프린터로 출력 | ⭐⭐⭐ |
| **🏢 공공기관 보안 친화** | 설치 불필요 = 망 분리 환경에서도 사용 가능 (인터넷망) | ⭐⭐⭐⭐ |
| **📋 기관별 템플릿 저장** | "우리 기관 기본 설정" 저장 → 다음부터 클릭만 | ⭐⭐⭐ |

> [!IMPORTANT]
> **킬러 기회:** "정부문서화일 전용 + 웹 기반 + 자동 텍스트 피팅"의 조합은 **시장에 없음**
